## 📊 Project Title
**Time Series Forecasting for Stock Market Using ARIMA, Prophet & LSTM**

## 🧠 Description
Forecasting the stock prices of **TCS (Tata Consultancy Services)** using:
- ARIMA (Model 1)
- Facebook Prophet (Model 2)
- LSTM Neural Network (Model 3)
